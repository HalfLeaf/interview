#  单词转换

##  算法题

::: tip 题目
给定字典中的两个词，长度相等。写一个方法，把一个词转换成另一个词， 但是一次只能改变一个字符

每一步得到的新词都必须能在字典中找到。

编写一个程序，返回一个可能的转换序列。

如有多个可能的转换序列，你可以返回任何一个
:::

~~~
示例1：
    输入:
        beginWord = "hit",
        endWord = "cog",
        wordList = ["hot","dot","dog","lot","log","cog"]
    输出:
        ["hit","hot","dot","lot","log","cog"]
~~~

~~~
示例2：
    输入:
        beginWord = "hit"
        endWord = "cog"
        wordList = ["hot","dot","dog","lot","log"]
    输出: []
~~~
